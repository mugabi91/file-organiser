Project Name

Organize Directory


Description

This Python script organizes a directory by moving files into subdirectories based on their file types.


Installation and Usage


Clone the repository to your local machine.

Open a terminal or command prompt and navigate to the cloned repository.

Run the script by executing the following command:


Copy code
Insert code
python organize_directory.py

Enter the path to the directory you want to organize when prompted.


Dependencies

This script requires the following dependencies:



Python 3.x

shutil module

os module
